## Téléchargement

Dernière version → onglet **[Releases](../../releases/latest)**.

| OS | Fichier |
|----|---------|
| **Windows** | `Upluce.Toolbox_x.y.z_x64-setup.exe` ou `_x64_en-US.msi` |
| **macOS** (Apple Silicon / Intel) | `Upluce.Toolbox_x.y.z_aarch64.dmg` / `_x64.dmg` |
| **Linux** | `Upluce.Toolbox_x.y.z_amd64.AppImage` ou `_amd64.deb` |

## Installation

- **Windows** : SmartScreen peut avertir (app non signée) → *Informations complémentaires* → *Exécuter quand même*.
- **macOS** : clic droit → *Ouvrir* au 1er lancement (app non signée).
- **Linux** : `.AppImage` (`chmod +x`) ou `.deb`.
