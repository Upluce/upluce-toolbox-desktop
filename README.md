# Upluce Toolbox

Boîte à outils développeur d'Upluce (gestion de secrets, connexions base de données, utilitaires), construite avec **Tauri 2**.

> Dépôt **public de distribution** : binaires uniquement. Code source dans `Upluce/upluce.com` (`apps/toolbox`).

## Téléchargement

Dernière version → onglet **[Releases](../../releases/latest)** : Windows (`.msi`/`.exe`), macOS (`.dmg` arm64 + x64), Linux (`.deb`/`.AppImage`).

## Mises à jour

Auto-updater Tauri (`latest.json`). Builds **v1.0 non signés**.
